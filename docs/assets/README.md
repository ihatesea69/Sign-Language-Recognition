# Assets Directory

This directory contains visual assets for documentation.

## Required Images/GIFs

To make the README complete, please add the following files:

### Demo Media
- `demo.gif` - Main demo showing the system in action
- `gesture_detection.png` - Screenshot of hand detection
- `gesture_recognition.png` - Screenshot of gesture recognition
- `text_to_speech.png` - Screenshot of TTS output

### Optional Assets
- `architecture_diagram.png` - System architecture diagram
- `workflow.png` - Workflow diagram
- Logo files

## Image Guidelines

- **Format:** PNG for screenshots, GIF for animations
- **Resolution:** Minimum 1280x720 for screenshots
- **File size:** Keep under 5MB
- **Compression:** Use optimized images

## How to Capture

### Demo GIF
1. Run the application
2. Use screen recording software (OBS, ShareX, ScreenToGif)
3. Perform various gestures
4. Convert to optimized GIF

### Screenshots
1. Run the application
2. Capture at key moments showing different features
3. Annotate if needed
4. Save as PNG
